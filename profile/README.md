中和高中 FRC 8569 北極星隊於2020年創立，目前邁入第四年。團隊由中和高中兩位生活科技教師帶領，參加各類機器人競賽，展現 FRC 競賽精神。學生積極參與募款活動，以支持競賽所需的資金，並與當地企業合作，獲得支持。2024年，團隊將前往美國加州參加 FRC 世界區賽，這是中和高中首次出國參賽，師生滿懷信心，期望在賽事中取得佳績。
